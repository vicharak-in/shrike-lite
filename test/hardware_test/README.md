# Test Bed for shrike hardware

### Follow these steps to use 

1. Clone the Shrike-lite Project 
```
git clone https://github.com/vicharak-in/shrike-lite.git
```

2. Change directory to the hardware_test dir in test dir

```
cd shrike-lite/test/hardware_test
```

3. Execute the setup file 

```
sudo chmod +x setup.sh
sudo ./setup.sh
```

4. Execute the test file 
```
sudo chmod +x test_shrike.sh
sudo ./test_shrike.sh
```

5. Connect the Boards One after another and you are good to Go
